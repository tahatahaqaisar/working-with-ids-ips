# Working With IDS/IPS

This repository contains my notes, labs, and learning progress from the Hack The Box module **Working with IDS/IPS**.

## About

In this module, I am learning the fundamentals of Intrusion Detection Systems (IDS) and Intrusion Prevention Systems (IPS), including how they are used in cybersecurity monitoring and threat detection.

## Topics Covered

- Introduction to IDS and IPS
- Network traffic monitoring
- Signature-based detection
- Rule creation and analysis
- Alert investigation
- Log analysis

## Tools & Technologies

### Suricata
- Understanding Suricata architecture
- Writing and analyzing Suricata rules
- Detecting malicious traffic
- Log and alert analysis

### Snort
- Snort rule syntax and structure
- Traffic inspection
- Intrusion detection techniques
- Creating custom detection rules

### Zeek
- Network security monitoring
- Zeek logs and scripting basics
- Traffic analysis and event monitoring
- Detecting suspicious activities

## Goals

- Build a strong foundation in IDS/IPS technologies
- Learn how network-based threat detection works
- Practice analyzing malicious network traffic
- Improve blue team and SOC analysis skills
